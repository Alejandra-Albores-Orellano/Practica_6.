¿Qué es el FRONTEND?

Es la parte visible de un sitio web o aplicación con la que los usuarios pueden interactuar directamente. Es la cara del sitio, ubicada en el lado del cliente.

¿Para qué sirve?

Es aquello que ven los usuarios e incluye elementos visuales, como botones, casillas de verificación, gráficos y mensajes de texto. Permite a los usuarios interactuar con la aplicación.

Componentes clave del frontend

  • Estructura (HTML) para crear el contenido
  • Estilo (CSS) para el diseño y presentación
  • Interacción (JavaScript) para la funcionalidad dinámica
  • Optimización para diferentes dispositivos (responsive design)
  • Uso de frameworks como Bootstrap y Materialize
  • Librerías JavaScript como jQuery y React
